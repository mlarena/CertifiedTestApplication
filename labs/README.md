# Лабораторные работы по PostgreSQL DBA

Набор из 45 лабораторных работ, основанных на курсе DBA1 «Администратор баз данных» (PostgreSQL Pro).

## Уровни

| Уровень | Кол-во | Фокус |
|---------|--------|-------|
| [Junior](junior/) | 15 | Базовые навыки: установка, psql, конфигурация, основы SQL |
| [Middle](middle/) | 15 | Архитектура: MVCC, VACUUM, WAL, статистика, производительность |
| [Senior](senior/) | 15 | Администрирование: бэкапы, репликация, безопасность, обновление |

## Структура каждой лабораторной

1. **Цель** — что изучаем
2. **Теория** — краткое описание концепции
3. **Задание** — пошаговые инструкции
4. **Решение** — готовые команды с выводом
5. **Контрольные вопросы** — для самопроверки
6. **Дополнительное задание** — усложнённый вариант (опционально)

## Требования

- PostgreSQL 16 на Ubuntu (или аналогичный Linux)
- Пользователь `student` с правами суперпользователя
- Утилита `psql`

## Порядок прохождения

Рекомендуется проходить работы последовательно: Junior → Middle → Senior. Внутри каждого уровня работы тоже идут по нарастающей сложности.
